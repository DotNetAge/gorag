# TimeExtractor

`TimeExtractor` 是 GoRAG 中用于提取时间信息的内置提取器。

## 功能

- 提取日期（如：2024年1月1日、2024-01-01）
- 提取时间（如：12:30、下午3点）
- 提取时间段（如：2024年1月-3月、上午9点到下午5点）
- 提取相对时间（如：昨天、明天、上周、下个月）
- 提取节日（如：春节、国庆节、圣诞节）

## 实现原理

基于 `PatternExtractor`，使用：

1. **日期正则**：识别各种日期格式
2. **时间正则**：识别各种时间格式
3. **时间段模式**：识别时间范围
4. **相对时间模式**：识别相对时间表达式

## 提取规则

| 规则     | 示例                            | 说明           |
| -------- | ------------------------------- | -------------- |
| 日期     | 2024年1月1日、2024-01-01        | 具体日期       |
| 时间     | 12:30、下午3点                  | 具体时间       |
| 时间段   | 2024年1月-3月、上午9点到下午5点 | 时间范围       |
| 相对时间 | 昨天、明天、上周、下个月        | 相对时间表达式 |
| 节日     | 春节、国庆节、圣诞节            | 节日名称       |

## 配置选项

| 参数              | 类型   | 说明             | 默认值          |
| ----------------- | ------ | ---------------- | --------------- |
| `IncludeRelative` | bool   | 是否包含相对时间 | true            |
| `IncludeHolidays` | bool   | 是否包含节日     | true            |
| `TimeZone`        | string | 时区设置         | "Asia/Shanghai" |

## 使用示例

```go
// 基本使用
timeExtractor := extractors.TimeExtractor

// 提取文本中的时间信息
text := "会议安排在2024年1月15日下午3点，持续到明天下午5点。"
entities, edges := timeExtractor.Extract(text)

// 结果：
// entities = [
//   {Type: "Time", Value: "2024年1月15日下午3点"},
//   {Type: "Time", Value: "明天下午5点"}
// ]
```

## 扩展方法

```go
// 添加自定义时间模式
timeExtractor.AddPattern("Time", `(\d{4})年(\d{1,2})月(\d{1,2})日`) // 中文日期格式

// 添加自定义相对时间
timeExtractor.AddPattern("Time", `(最近|近期|近期内|不久前|很快)`) // 相对时间表达式

// 添加自定义节日
timeExtractor.AddPattern("Time", `(元旦|春节|清明节|劳动节|端午节|中秋节|国庆节)`) // 中国节日
```
