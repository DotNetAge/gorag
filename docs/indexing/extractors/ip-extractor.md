# IPExtractor

`IPExtractor` 是 GoRAG 中用于提取 IP 地址的内置提取器。

## 功能

- 提取 IPv4 地址（如：192.168.1.1、10.0.0.1）
- 提取 IPv6 地址（如：2001:0db8:85a3:0000:0000:8a2e:0370:7334）
- 提取带端口的 IP 地址（如：192.168.1.1:8080、[2001:0db8::1]:8080）

## 实现原理

基于 `RegexExtractor`，使用：

1. **IPv4 正则**：识别 IPv4 地址格式
2. **IPv6 正则**：识别 IPv6 地址格式
3. **端口模式**：识别带端口的 IP 地址

## 提取规则

| 规则 | 示例 | 说明 |
|------|------|------|
| IPv4 地址 | 192.168.1.1、10.0.0.1 | 标准 IPv4 地址 |
| IPv6 地址 | 2001:0db8:85a3:0000:0000:8a2e:0370:7334、::1 | 标准 IPv6 地址 |
| 带端口的 IP | 192.168.1.1:8080、[2001:0db8::1]:8080 | 带端口号的 IP 地址 |

## 配置选项

| 参数 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| `IncludeIPv4` | bool | 是否包含 IPv4 地址 | true |
| `IncludeIPv6` | bool | 是否包含 IPv6 地址 | true |
| `IncludePort` | bool | 是否包含带端口的 IP | true |

## 使用示例

```go
// 基本使用
ipExtractor := extractors.IPExtractor

// 提取文本中的 IP 地址
text := "服务器地址为 192.168.1.1:8080，IPv6 地址为 2001:0db8::1"
entities, edges := ipExtractor.Extract(text)

// 结果：
// entities = [
//   {Type: "IP", Value: "192.168.1.1:8080"},
//   {Type: "IP", Value: "2001:0db8::1"}
// ]
```

## 扩展方法

```go
// 添加自定义 IPv4 模式
ipExtractor.AddPattern("IP", `(?:(?:25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)\.){3}(?:25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)`) // IPv4 地址

// 添加自定义 IPv6 模式
ipExtractor.AddPattern("IP", `(?:[0-9a-fA-F]{1,4}:){7}[0-9a-fA-F]{1,4}`) // IPv6 地址

// 添加带端口的 IP 模式
ipExtractor.AddPattern("IP", `(?:(?:25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)\.){3}(?:25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?):\d{1,5}`) // 带端口的 IPv4
```
