# PhoneExtractor

`PhoneExtractor` 是 GoRAG 中用于提取电话号码的内置提取器。

## 功能

- 提取中国手机号（如：138-0000-0000、13912345678）
- 提取中国固定电话（如：010-12345678、021-87654321）
- 提取国际电话号码（如：+1-123-456-7890、+86-138-0000-0000）
- 提取传真号码（如：FAX: 010-12345678）

## 实现原理

基于 `RegexExtractor`，使用：

1. **手机号正则**：识别中国手机号格式
2. **固定电话正则**：识别固定电话格式
3. **国际电话正则**：识别国际电话号码格式
4. **传真号码模式**：识别传真号码

## 提取规则

| 规则 | 示例 | 说明 |
|------|------|------|
| 中国手机号 | 138-0000-0000、13912345678 | 11位手机号 |
| 中国固定电话 | 010-12345678、021-87654321 | 区号+电话号码 |
| 国际电话 | +1-123-456-7890、+86-138-0000-0000 | 国家代码+电话号码 |
| 传真号码 | FAX: 010-12345678 | 传真标识+电话号码 |

## 配置选项

| 参数 | 类型 | 说明 | 默认值 |
|------|------|------|--------|
| `IncludeFax` | bool | 是否包含传真号码 | true |
| `IncludeInternational` | bool | 是否包含国际电话号码 | true |

## 使用示例

```go
// 基本使用
phoneExtractor := extractors.PhoneExtractor

// 提取文本中的电话号码
text := "联系电话：138-0000-0000，固定电话：010-12345678"
entities, edges := phoneExtractor.Extract(text)

// 结果：
// entities = [
//   {Type: "Phone", Value: "138-0000-0000"},
//   {Type: "Phone", Value: "010-12345678"}
// ]
```

## 扩展方法

```go
// 添加自定义电话模式
phoneExtractor.AddPattern("Phone", `1[3-9]\d{9}`) // 中国手机号

// 添加自定义固定电话模式
phoneExtractor.AddPattern("Phone", `0[1-9]\d{1,2}-?\d{7,8}`) // 中国固定电话

// 添加自定义国际电话模式
phoneExtractor.AddPattern("Phone", `\+\d{1,4}-?\d{3,4}-?\d{3,4}-?\d{3,4}`) // 国际电话
```
