## ⏳ Phase 4: 使用 AgenticMetadata 重构所有 Step

### 需要修改的 Step Execute 方法

**当前模式**（黑板反模式）：
```go
func (s *intentRouter) Execute(...) error {
    result, _ := s.classifier.Classify(ctx, state.Query)
    state.Query.Metadata["intent"] = string(result.Intent)  // ❌ 魔术字符串
    return nil
}
```

**新模式**（强类型）：
```go
func (s *intentRouter) Execute(...) error {
    result, _ := s.classifier.Classify(ctx, state.Query)
    
    if state.Agentic == nil {
        state.Agentic = retrieval.NewAgenticMetadata()
    }
    state.Agentic.Intent = string(result.Intent)  // ✅ 强类型字段
    
    // 可选：同步到 Query.Metadata 以保持向后兼容
    state.Agentic.MergeToQuery(state.Query)
    
    return nil
}
```
