    自定义 Parser¶

    概述¶

    通过实现 Parser 接口，创建自定义文档解析器。

    接口要求¶

    type Parser interface {
    Parse(ctx context.Context, reader io.Reader) ([]*Document, error)
}

    实现步骤¶

    1. 定义结构体¶

    type myParser struct {
    // 自定义配置
    options map[string]any
}

    2. 实现 Parse 方法¶

    func (p *myParser) Parse(
    ctx context.Context,
    reader io.Reader,
) ([]*core.Document, error) {
    // 1. 读取数据
    data, err := io.ReadAll(reader)
    if err != nil {
        return nil, err
    }

    // 2. 解析数据
    docs := make([]*core.Document, 0)

    // 自定义解析逻辑
    content := p.extractContent(data)

    doc := &core.Document{
        ID:      generateID(),
        Content: content,
        Metadata: map[string]any{
            "source": "custom",
        },
    }

    docs = append(docs, doc)

    return docs, nil
}

    3. 创建构造函数¶

    func NewMyParser(options map[string]any) *myParser {
    return &myParser{
        options: options,
    }
}

    使用示例¶

    indexer := indexer.NewBuilder().
    WithParser(NewMyParser(nil)).
    Build()

file, _ := os.Open("document.xyz")
defer file.Close()

indexer.Index(ctx, file)

    测试建议¶

    func TestMyParser_Parse(t *testing.T) {
    parser := NewMyParser(nil)

    reader := strings.NewReader("测试内容")

    docs, err := parser.Parse(ctx, reader)

    assert.NoError(t, err)
    assert.NotEmpty(t, docs)
}

    注意事项¶

    1. 流式处理：大文件建议流式解析
    2. 错误处理：解析失败要正确返回
    3. 元数据：提取有用的元数据信息
